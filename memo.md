# Design memo

## Call graph

```
+-----+   +------+
| shp |-->| shpc |
+-----+   +------+
   |
   |      +------------+   +----------+
   +----->| shprun.tcl |-->| shpfront |
          +------------+   +----------+
                 |
                 |         +---------------+
                 +-------->| compiled code |
                           +---------------+
```

### Compilation

```
/--------------\   +------+   /------------\
| Sheep script |-->| shpc |-->| Tcl script |
\--------------/   +------+   \------------/
```

```
/------------\   +----------+   /-------------\   +--------------------------+
| Input data |-->| shpfront |-->| MessagePack |-->| shprun.tcl/compiled code |
\------------/   +----------+   \-------------/   +--------------------------+
                                                              |
                                                              v
                                                      /-------------\
                                                      | Output data |

```
